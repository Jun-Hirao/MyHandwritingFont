## アプリ名: MyHandwritingFont

## アプリの仕様
**概要**
自分の手書き文字をフォントに変換するアプリです。ペンタブやマウスで文字を書いて、それをデジタルフォント（.ttf形式）に変換することで、手書き風の履歴書などに利用できます。対応OSはWindowsです。

## 開発環境
- Windows11(23H2)
- pyenv-win
- Python 3.11.9
- venv
